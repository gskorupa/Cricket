# Cricket Microservices Framework for Java.


## Building microservices

Add dependency to your pom.xml

```xml
<dependency>
    <groupId>org.cricketmsf</groupId>
    <artifactId>cricket</artifactId>
    <version>2.0.0-rc36</version>
</dependency>
```
