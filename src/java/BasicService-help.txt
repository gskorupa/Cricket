
Usage: java -jar cricketms-1.2.38.jar [options]

Runs selected service class.

Options:
--help | -h           Print this usage information and exit
--run | -r            Run the configured service on the embeded http server
--config | -c PATH    Use external configuration file at PATH
--service | -s ID     Run selected service defined in the configuration file 
--print | -p          Print configuration and exit
--lift | -l ServiceClass   Run specified class as a service (lift mode)

Without -r option, the runOnce() method of default BasicService service will be executed.
